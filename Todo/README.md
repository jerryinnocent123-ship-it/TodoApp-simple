# Todo App - Gestionnaire de tâches moderne

une application Todo complète construite avec React et Tailwind CSS, permettant de gérer vos tâches quotidiennes avec une interface moderne et intuitive.
# Fonctionnalités

## Gestion des tâches
-  **Ajouter des tâches** - Créez de nouvelles tâches avec titre et description
-  **Modifier des tâches** - Mettez à jour vos tâches existantes
- **Supprimer des tâches** - Supprimez les tâches terminées ou inutiles
-  **Marquer comme complété** - Indiquez les tâches accomplies avec un simple clic
-  **Sauvegarde automatique** - Toutes vos tâches sont sauvegardées dans le localStorage du navigateur

### Interface utilisateur
-  **Design moderne** - Interface épurée avec Tailwind CSS
-  **Responsive** - S'adapte parfaitement à tous les appareils (mobile, tablette, desktop)
-  **Horloge en temps réel** - Affichage dynamique de la date et l'heure
-  **Feedback visuel** - Animations et transitions pour une expérience utilisateur fluide
-  **Thème coloré** - Dégradé bleu/violet pour l'en-tête
### Expérience utilisateur
- **Validation des formulaires** - Empêche l'ajout de champs vides
- **Compteur visuel** - Visualisation claire des tâches complétées
- **Persistance des données** - Les tâches restent après rafraîchissement de la page
- **Actions rapides** - Icônes intuitives pour toutes les actions

##  Technologies utilisées
- **React 18** - Framework JavaScript pour l'interface utilisateur
- **Tailwind CSS** - Framework CSS pour le styling
- **Lucide React** - Bibliothèque d'icônes modernes
- **UUID** - Génération d'identifiants uniques
- **LocalStorage** - Stockage persistant des données
